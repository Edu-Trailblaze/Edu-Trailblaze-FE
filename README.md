## ✨ Overview

**Edu Trailblaze** is a modern e-learning platform inspired by Udemy & Coursera, featuring a robust tech stack, beautiful UI, and smart features for students, teachers, and admins.

## 🌟 Features

| Role              | Capabilities                                                                     |
|-------------------|----------------------------------------------------------------------------------|
| 🧑‍🎓 **Students**  | Browse courses, enroll, track progress, get AI-powered recommendations           |
| 🧑‍🏫 **Teachers**  | Create, edit, and manage courses with a simple, elegant interface                |
| 🛡️ **Admins**     | Full platform management: users, courses, settings, and analytics                |

- 🔒 **Authentication & Authorization:** NextAuth (OAuth, social logins), role-based access.
- 🖥️ **Fully Responsive UI:** Lightning-fast, mobile-first design with modern UX.
- 💳 **Secure Payments:** Integrated payment (Stripe).
- 🤖 **AI-Powered:** Personalized course suggestions using ML.
- ⚡ **Performance:** SSR/SSG, lazy loading, best practices.

  NextAuth (OAuth, Google, Facebook, etc.), Firebase (optional)
- **Payment:**  
  Stripe
- **Dev Tools:**  
  ESLint, Prettier, Husky

## 🚀 Getting Started

1. **Clone the repository:**
    ```sh
    git clone https://github.com/Edu-Trailblaze/Edu-Trailblaze-FE.git
    cd Edu-Trailblaze-FE
    ```

2. **Install dependencies:**
    ```sh
    npm install
    ```

3. **Set up environment variables:**  
   Copy `.env.example` to `.env.local` and update values as needed.

4. **Run the development server:**
    ```sh
    npm run dev
    ```

5. **Open in browser:**  
   Go to [http://localhost:3000](http://localhost:3000)
